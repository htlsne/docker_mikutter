# What's This
TEOKURE condensed container within [mikutter](https://mikutter.hachune.net)

# How to Use
```
$ docker pull orumin/mikutter:latest
$ docker run -e DISPLAY=$DISPLAY -v /tmp/.X11-unix/:/tmp/.X11-unix/ orumin/mikutter:latest
```
